## Реалізація
У проекті створено два класи:
* **SportsEquipment**: описує сутність товару. Містить поля `name`, `brand`, `price`, `weight` та `stock`. Реалізовано методи `equals()` та `hashCode()` для коректного порівняння об'єктів, а також `toString()` для форматованого виводу.
* **Lab3Main**: містить виконавчий метод `main`, де ініціалізується масив, проводиться сортування за допомогою `Comparator` та виконується пошук об'єкта.

## Приклад роботи
Програма виводить список товарів, відсортований за ціною. Якщо ціна однакова (як у випадку з гантелями по 800.0), вони сортуються за вагою від більшої до меншої.
